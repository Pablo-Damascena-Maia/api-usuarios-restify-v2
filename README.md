📘 API de Usuários com Restify e Prisma

Projeto desenvolvido para a disciplina Programação Web II, com o objetivo de implementar uma API REST simples utilizando Node.js, Restify e Prisma ORM.

🚀 Tecnologias utilizadas
Node.js
Restify
Prisma ORM
MySQL
Nodemon
Dotenv
